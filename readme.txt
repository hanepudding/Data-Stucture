SSE.Simulator.exe是一个模拟程序，它模拟了真实投资环境。
你将获得100万元的初始资金并连续操作10日。
10日结束后，你将看到本次投资的收益率。
你的买卖操作将被记录至record.txt。
在操作过程中，你不可以超出余额购买股票，但你可以卖空！要问为什么的话，这就是自由市场！

HuffmanCode.exe是二叉树编码程序。
它的运行需要origin.txt来构建一棵哈夫曼树，并依据该哈夫曼树对01code.txt中的01编码进行解码。
解码结果将输出至decode.txt。

Encrypt.exe是加密/解密程序。
它的运行需要key.txt来存储密钥，并根据密钥对input.txt中的文本进行加密/解密，结果将保存在output.txt中。
明文运行本程序后将得到密文，反之亦然。要问这是为什么，这就是科学！
